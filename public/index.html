⚙️ Veriff Setup Instructions (for developers)

1. Create a free Veriff account
Go to portal.veriff.com → sign up → create an Integration.

2. Copy your API keys
From the integration page, copy API Key and Shared Secret Key.

3. Add environment variables in Render
In your Render service → Environment → add:

VERIFF_API_KEY = your_api_key_here VERIFF_SECRET = your_shared_secret_here APP_URL = https://tracker-app-qeoi.onrender.com

4. Configure webhook in Veriff portal
Under your Integration → Notifications → set the webhook URL to:
https://tracker-app-qeoi.onrender.com/api/kyc/webhook

5. Test with Veriff's sandbox
Veriff provides magic test documents in sandbox mode. Any scan will return "approved" or specific decline scenarios so you can test the full flow without a real ID.

Greek document support
Veriff fully supports: Αστυνομική Ταυτότητα (ID_CARD) and Διαβατήριο (PASSPORT). Liveness check (selfie match) is included automatically in every verification session.
